Project Overview

This project analyzes supply chain operations for GlamourGlow Cosmetics using Tableau. The dashboard provides insights into inventory management, supplier performance, shipping efficiency, and cost structure to support data-driven decision-making.

Interactive Filters
Shipping Carrier
Supplier Name
Product Type
Key Insights
Inventory imbalances highlight opportunities to reduce stockouts and excess inventory.
Supplier performance varies significantly in terms of lead times and defect rates.
Carrier B demonstrates the fastest average shipping time.
Operational costs represent the largest share of supply chain expenses.
